# 💰 Assistente Financeiro Educacional com IA

Este projeto foi desenvolvido como parte de um desafio da **DIO (Digital Innovation One)**. Trata-se de um Assistente Financeiro interativo que utiliza Inteligência Artificial para ajudar iniciantes a entenderem conceitos de economia e simularem investimentos de forma educacional.

## 🚀 Funcionalidades
* **Chatbot Inteligente:** Responde a dúvidas financeiras utilizando a API do Google Gemini.
* **Simulador de Juros Compostos:** Uma barra lateral interativa (Sidebar) onde o usuário pode arrastar valores para simular o crescimento do seu dinheiro ao longo do tempo.
* **Perguntas Rápidas (Fast Prompts):** Botões de atalho com as dúvidas mais comuns de iniciantes.
* **Guardrails (Segurança):** A IA foi instruída por *System Prompt* a nunca recomendar ativos de renda variável e a manter o foco estritamente em educação financeira.

## 🛠️ Tecnologias Utilizadas
* **Python** (Linguagem principal)
* **Streamlit** (Criação da interface web e UX)
* **Google Generative AI / Gemini 2.5 Flash** (Motor do chatbot)
* **Python-dotenv** (Proteção de variáveis de ambiente e chaves de API)
